ScrabScrap konfigurieren

Mit dieser Anwendung kann ScrabScrap eingestellt und konfiguriert werden. Über das Menü können die verschiedenen Funktionen zur Konfigurartion gestartet werden.

Sobald alle Einstellungen vorgenommen wurden, kann über den Menüpunkt "Start App" wieder ScrabScrap aufgerufen werden. Dazu wird der Rechner neu gestartet.

Hinweis: Solange diese Anwendung aktiv ist, wird der ScrapScrap-HotSpot genutzt. In diesem Modus ist kein Internet Zugriff möglich.
